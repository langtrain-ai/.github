## About

LangTrain democratizes AI training with intuitive, production-ready tools.  
It turns complex fine-tuning workflows into efficient, elegant experiences.  
Built for developers, researchers, and enterprises alike.

---

## Products

### LangTune  
Specialized text model training with LoRA fine-tuning.  
Supports conversational AI, domain adaptation, and multi-model compatibility.

### LangVision  
Multimodal vision-language training for image captioning, visual QA, and scene understanding.

---

## Quick Start

```bash
pip install langtrain-ai
````

```python
import langtrain as lt

trainer = lt.create_trainer("gpt2", "your_data.json")
trainer.train()
```

## Key Features

* Memory efficient: up to 95% less memory usage with LoRA
* Multi-modal: unified platform for text and vision models
* Production ready: scalable and reliable
* Developer friendly: simple APIs, clear documentation
